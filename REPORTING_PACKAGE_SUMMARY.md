## Step-By-Step Usage

### Phase 1: Run Evaluations (In Colab)
```
1. Run Notebook 1 → Datasets generated
2. Run Notebook 2 → Evaluation results (results/raw/*.json)
3. Run Notebook 3 → Report charts (results/reports/*.png)
```

### Phase 2: Generate LaTeX Report
```bash
# From project root
python generate_report.py
# Output: results/reports/VLM_ARB_Report_<timestamp>.tex
```

### Phase 3: Insert Charts
1. Open generated `.tex` file
2. Find `\textit{[INSERT FIGURE:`...`]}`
3. Replace with: `\includegraphics[width=0.9\textwidth]{./01_model_comparison.png}`
4. Repeat for other charts

### Phase 4: Generate PDF
```bash
# Make sure LaTeX is installed
pdflatex results/reports/VLM_ARB_Report_*.tex
# Output: results/reports/VLM_ARB_Report_*.pdf
```

## Customization

### Add/Change Metrics
If your Notebook 2 computes different metrics, edit `generate_report.py`:
```python
# In _generate_results_table()
for model_name in ['clip', 'mobilevit', 'blip2', 'llava']:
    # Modify extraction logic
    asr = m.get('your_metric_name', 0)
```

### Change Report Title
Edit in `VLM_ARB_Report.tex`:
```latex
\title{\textbf{\Large VLM-ARB: Your Custom Title}}
```

### Add New Sections
```latex
\section{My New Section}

Your content here...

\subsection{Subsection}
```

### Modify Styling
The template uses:
- `\textbf{}` for bold
- `\textit{}` for italics
- `\textcolor{darkblue}{}` for colors
- `\section{}` for headings
- `$...$ `for inline math
- `$$...$$` for display math
