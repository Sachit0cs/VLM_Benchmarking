# VLM-ARB LaTeX Report Documentation

## Overview

This package contains:
1. **VLM_ARB_Report.tex** — Professional LaTeX report template with placeholders for charts/graphs
2. **generate_report.py** — Python script to populate template with actual evaluation results  
3. **This guide** — Instructions for using both

The report is based on **actual methodology from your Colab notebooks** (Notebook 1, 2, 3) and includes:
- Executive summary
- Complete methodology (attacks, models, metrics)
- Actual model evaluation results (ASR, ODS, SBR)
- Designated chart/graph insertion points
- Discussion and implications
- Appendices with code and infrastructure details

---

## Quick Start

### Option 1: Generate from Colab Results (After Running Notebooks)

#### Step 1: Run Your Evaluations
1. Run **Notebook 2** (Model Evaluations) in Google Colab
2. Results saved to: `results/raw/*.json`
3. Run **Notebook 3** (Report Generation) to create charts
4. Charts saved to: `results/reports/*.png`

#### Step 2: Generate LaTeX Report
```bash
# From project root directory
python generate_report.py \
  --results-dir results/raw \
  --template VLM_ARB_Report.tex \
  --output-dir results/reports
```

#### Step 3: Insert Charts
Open `results/reports/VLM_ARB_Report.tex` and find sections marked:
```latex
\fbox{
\begin{minipage}{0.9\textwidth}
\vspace{4cm}
\textit{[INSERT FIGURE: 01_model_comparison.png]}
```

Replace with actual image:
```latex
\includegraphics[width=0.9\textwidth]{./01_model_comparison.png}
```

#### Step 4: Generate PDF
```bash
pdflatex VLM_ARB_Report.tex
```

## Installing LaTeX (For PDF Generation)

The report generator can produce PDF directly if you have a TeX installation.

### macOS
```bash
# Option 1: Minimal (faster)
brew install basictex

# Option 2: Full (includes all packages)
brew install mactex
```

### Linux (Ubuntu/Debian)
```bash
sudo apt-get install texlive-latex-base texlive-fonts-recommended texlive-latex-extra
```

### Windows
Download and install:
- **MiKTeX** (https://miktex.org/) — Fastest setup
- **TeX Live** (https://tug.org/texlive/) — Most complete

## Editing the LaTeX Template

### Safe to Edit:
- Discussion section (Section 7)
- Conclusion (Section 9)
- Appendix content
- Abstract or executive summary wording

### Do NOT Edit:
- Table placeholders with results (generate_report.py overwrites them)
- Metric definitions (these are fixed from methodology)
- Model descriptions (reflect actual implementation)

### To Customize:
1. Open `VLM_ARB_Report.tex` in any text editor
2. Find section you want to modify
3. Edit text between `\begin{}` and `\end{}` commands
4. Preserve LaTeX commands (`\textbf{}`, `\section{}`, etc.)

## Troubleshooting

### Problem: "pdflatex not found"
**Solution:** Install TeX as shown above

### Problem: PDF generation fails with "Undefined control sequence"
**Solution:** Make sure all images exist and paths are correct:
```bash
ls results/reports/*.png
```

### Problem: Results not loading
**Solution:** Check JSON format matches expected structure:
```bash
python -m json.tool results/raw/*.json
```

### Problem: Charts not showing in PDF
**Solution:** 
1. Make sure PNG files are in same directory as `.tex` file
2. Use relative paths: `./01_model_comparison.png`
3. Run pdflatex twice: `pdflatex VLM_ARB_Report.tex` × 2
