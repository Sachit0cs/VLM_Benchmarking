# Quick start guide for the Streamlit UI

## 🚀 Start in 30 Seconds

### macOS / Linux:
```bash
chmod +x run_ui.sh
./run_ui.sh
```

### Windows:
```bash
pip install streamlit
python -m streamlit run ui/app.py
```

That's it! Browser opens automatically.

---

## 📋 What You Get

A professional web dashboard with 5 tabs:

1. **🚀 Quick Demo** - See attacks in action (30 sec demo)
2. **🎯 Single Attack** - Try individual attacks
3. **📊 Batch Evaluation** - Run full evaluation pipeline
4. **📈 Results Analysis** - View comparative metrics
5. **📋 Project Info** - Overview & stats
